# REMNANT

Prototipo de videojuego 2D desarrollado en Unity.

## Descripción
REMNANT es un videojuego tipo top-down shooter donde el jugador debe explorar, completar objetivos y enfrentarse a enemigos para avanzar entre zonas.

## Mecánicas principales
- Movimiento en 4 direcciones (WASD)
- Disparo en dirección al mouse
- Sistema de misiones
- Interacción con objetos (radio, sensores, terminales)
- Enemigos con IA básica
- Sistema de vida y daño
- Transiciones entre escenas

## Controles
- WASD: Movimiento
- Mouse: Apuntar
- Click: Disparar
- F: Interactuar

## Tecnologías usadas
- Unity 6
- C#
